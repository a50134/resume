# Chenghsin Lee - Professional Portfolio

一個展示 Chenghsin Lee 職業背景、專案經驗和技能的專業個人作品集網站。

## 🎯 特色

- **現代設計**：採用藍色主題 (#6495ED)，專業且優雅
- **響應式設計**：完美適配桌面、平板和手機
- **流暢動畫**：使用 AOS (Animate On Scroll) 實現頁面動畫
- **完整內容**：展示職業背景、技能、專案和聯絡方式

## 📋 頁面結構

- **首頁 (Hero Section)**：個人介紹和號召行動
- **關於我**：職業背景和核心特質
- **專案展示**：DoraHandmade 和 I-need-this 專案
- **技能與語言**：技術技能、語言能力和其他技能
- **聯絡**：聯絡方式和社群媒體

## 🛠️ 技術棧

- **HTML5**：語義化標記
- **CSS3**：自訂樣式和響應式設計
- **JavaScript**：互動功能和動畫
- **Bootstrap 5**：UI 框架
- **AOS**：滾動動畫庫
- **Vite**：前端構建工具

## 📦 安裝與運行

### 安裝依賴
```bash
npm install
```

### 開發模式
```bash
npm run dev
```

### 生產構建
```bash
npm run build
```

### 預覽構建結果
```bash
npm run preview
```

## 🎨 自訂選項

### 修改主色彩
編輯 `src/css/style.css` 中的 CSS 變數：
```css
:root {
    --primary-color: #6495ED;
    --primary-dark: #4169E1;
    --primary-light: #87CEEB;
}
```

### 修改內容
編輯 `index.html` 中的相應部分即可更新：
- 個人信息
- 專案描述
- 技能清單
- 聯絡方式

## 📱 響應式斷點

- **平板**：768px - 1199px
